## `@plexis/toSwapCase`

Swaps alphabet case from a string

```javascript
import toSwapCase from '@plexis/to-swap-case';

toSwapCase('Hello WORLD');
// => 'hELLO world'

toSwapCase('123 toys');
// => '123 toys'
```

**Aliases**

```javascript
import toSwapCase from '@plexis/to-swap-case';
import {toSwapCase, swapCase} from 'plexis';
```
